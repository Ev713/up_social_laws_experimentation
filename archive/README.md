# Archive

This directory contains generated or historical data that is not part of the
active experiment/test layout.

- `runs/`: older experiment run outputs moved out of `experimentation/runs/`.
- `scripts/legacy_experiment_helpers/`: old standalone helper scripts replaced by `python -m experimentation.cli`.
- `test_logs/`: historical logs and CSVs moved out of the old `test/` folder.
- `experimentation/runs_backup/`: previous backup copy from `experimentation/.runs_backup/`.

Active experiment outputs remain under `experimentation/runs/`.
